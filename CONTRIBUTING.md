## How to Install 

**In command line:**

Make sure python3 is installed:
```bash
python3 --version
```
Make sure tomli is downloaded:
```bash
pip3 install tomli
```

Clone the Repo

```bash
git clone https://github.com/mingming-ma/txt2html.git
cd txt2html 
```

Make the tool executable
```bash
chmod +x txt2html.py
```
